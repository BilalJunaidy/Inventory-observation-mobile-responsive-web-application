Inventory listing upload instruction

Please comply with the following requirements before submitting this form:

  1. The uploaded inventory listing should be an excel file (either an xlsx, an xlsm, xls or a csv).
  2. Please ensure that the entirety of the inventory listing uploaded is on the first tab of the uploaded file.
  3. The required fields for this form include:
    1. The row number in the uploaded file for the column headers
    2. The row number in the uploaded file for where the inventory listing starts
    3. The SKU/Product Code which helps uniquely identify the specific product
    4. The inventory quantity on hand of the SKU/Product code
